## |- Телеграмм бот тех поддержки -|

### | - Скачивание и настройка -|
Для использования бота загрузите репозиторий расположенный по ссылке 
https://github.com/GregoryMelikhovGit/techsupport-bot, затем, после загрузки
извлеките файлы в предложенную папку. После, откройте VSCode и выберите
файл -> открыть папку  и выберите папку в которую вы извлекли проект.
Далее откройте файл config.py и вставьте в одноименную переменную свой Телеграмм
токен, сохраните файл. 

### |- Запуск бота -|
Для запуска бота вам понадобится запустить три файла по очереди:
config.py -> logic.py -> main.py, и бот готов к использованию!
